HIVE

HIVE collaborative code editor was created with instructors and students in mind. With real time coding and syntax highlighting, you can watch your instructor code on your own screen and post up questions on the question board for your instructor to see so they can answer it during the lesson!

Built with:

  • Firebase
  • Firepad
    • CodeMirror
  • Express
  • jQuery
  • Node.JS
  • Bootstrap

HashRobot

HashRobot is your personal marketing consultant. HashRobot will give the most visible and used hashtags in a given category. Using these hashtags will give social media users an edge in gaining more visibility and followers. HashRobot makes it easy for you to tweet right from our hashdash. Great content deserves to be #noticed.

Built with:

  • Ruby on Rails
  • Materialize.css
  • Font Awesome
  • Google Fonts
  • Heroku
  • Twitter API
    • To display hashtags and OAuth
  • MonkeyLearn API
    • To analyze tweets and retrieve relevant categories
